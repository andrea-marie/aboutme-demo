HTML

A web page is a simple text file written in HTML (Hypertext Markup Language). A Hypertext Markup Language is a language that supports both the tagging of distinct document elements and connecting documents through hypertext links.

CSS

The appearance of the HTML page is determined by one or more style sheets written in the Cascading Style Sheets (CSS) language.

Javascript

The programming language for client-side programs is JavaScript. JavaScript is an interpreted language, meaning that the program code is executed directly without requiring an application known as a compiler to translate the code into machine language. You need only two things to use JavaScript: a text editor to write the JavaScript code and a browser to run the commands.
